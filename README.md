# NodeJS_CLI_Tool
pwndmail is an email breach checker.

## Install
``$ npm i -g pwndmail``

## How to use it
``$ pwndmail email@to.check``
## API
[Have I been pwned?](https://haveibeenpwned.com/API/v2 "Have I been pwned")
